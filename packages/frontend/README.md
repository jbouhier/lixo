# Frontend ♻️

## Pnpm Commands
- `dev` - Run local server
- `build` - Build for prodution
- `lint` - Linter
- `test` - Tests
- `test:ui` - Tests with UI
- `coverage` - Test coverage `./coverage/index.html` (open in browser)
- `json` - Mock API server
